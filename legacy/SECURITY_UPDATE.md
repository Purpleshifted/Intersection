# 보안 업데이트

## CVE-2025-66478 패치 완료

Next.js 16.0.3에서 발견된 심각한 보안 취약점(CVE-2025-66478)을 패치했습니다.

### 업데이트 내용
- **Next.js**: `16.0.3` → `16.0.7` ✅
- **eslint-config-next**: `16.0.3` → `16.0.7` ✅

### 취약점 정보
- **심각도**: 높음
- **유형**: 원격 코드 실행 (RCE)
- **영향**: React Server Components (RSC) 프로토콜의 안전하지 않은 역직렬화

### 조치 사항
1. ✅ Next.js를 16.0.7로 업그레이드 완료
2. ✅ eslint-config-next도 동일 버전으로 업데이트 완료
3. ✅ GitHub에 푸시 완료

### 다음 단계
- Vercel에 배포된 경우 자동으로 재배포됩니다
- 로컬에서 개발하는 경우: `yarn install` 실행
